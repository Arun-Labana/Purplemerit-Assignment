Tech Stack
Languages: JavaScript/TypeScript or Python (both is a bonus)

Frontend: React + Next.js (App Router), Tailwind/shadcn, Monaco Editor

Backend: Node.js (Express/Fastify) or FastAPI; REST + WebSockets

Backend Development Role Description
Design, develop, and maintain scalable, high-performance server-side applications and APIs integrated with front-end and third-party systems.
Build and maintain RESTful and microservice APIs that support internal and external integrations.
Implement and manage data storage solutions, including relational and non-relational databases, ensuring efficiency and data integrity.
Write and execute unit and integration tests (e.g., JUnit, Mocha). Debug, identify, and fix performance bottlenecks.
Monitor emerging technologies, frameworks, and backend trends to continuously enhance performance and architecture.
Ensure all deliverables meet production standards, including security, scalability, documentation, and deployment readiness.
Core Requirements
1–4 years of proven backend development experience in production environments.
Strong proficiency in JavaScript and/or Node.js.
Expertise with both relational databases (MySQL, PostgreSQL) and non-relational databases (MongoDB, Cassandra, Redis).
Deep understanding of microservices architecture and API-first design principles.
Experience deploying applications on AWS, Azure, or GCP, with working knowledge of CI/CD pipelines and containerization (Docker, Kubernetes).
Proficiency in API security protocols (OAuth, JWT) and general web application security practices.
Strong understanding of Git workflows, including versioning, branching, and merging.
Experience writing automated tests using frameworks such as JUnit, Mockito, or Mocha.
Knowledge of caching mechanisms (Redis, Memcached), load balancing, and horizontal scaling techniques.
Experience with message brokers like RabbitMQ, Kafka, or AWS SQS for asynchronous communication.
Familiarity with Serverless architectures (AWS Lambda, Azure Functions).
